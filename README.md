# Chayma Chouikh — BI Student & Data Analyst Portfolio

Professional blue, black and white portfolio built with Next.js.

## Run locally

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Before deployment

Replace the placeholders:

- `https://your-domain.com` in metadata, sitemap and JSON-LD
- `your-email@example.com` in the contact page
- Add your real CV file if you want a download button
- Replace SVG placeholders with your real project screenshots if available
